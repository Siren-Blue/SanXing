# 吾日三省吾身——日程管理小程序

- 效果：根据日程待办事项清单实时生成电脑/手机壁纸

- 原理：参考TextBlob中WordCloud的生成过程。

- 流程：用户输入待办事项$\rightarrow$根据清单生成图$\rightarrow$删除之前的图像$\rightarrow$将新图片设为壁纸。

- 计划实现的自定义内容：
  
  - 优先度：根据日期/自定义重要性决定字体大小
  
  - 配色
  
  - 字组成的形状
  
  - 字体
  
  - 中心字，如“吾日三省吾身”，“别摸鱼了快学习！”
  
  - 句式，如“xxx做了吗？”，“你是否忘了大明湖畔的xxx”，etc。
